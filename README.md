# 魔法文具工坊 / Magical Stationery Store

> 一款结合 **类吸血鬼幸存者 + 搜打撤 + 模拟经营** 的像素风动作经营游戏。  
> 白天经营文具店，夜晚深入亚空间，用“文具魔法”对抗由现实损耗具象化而成的侵蚀怪物。

---

## 📌 项目简介

**《魔法文具工坊》** 是一个以“文具店”为核心题材的原创游戏项目。  
玩家将扮演主角 **温姬妤**，在蔚兰群岛接手一家文具店，白天进行售卖、整理、补货与经营，夜晚则进入隐藏在现实背后的 **亚空间**，使用各种魔法文具对抗侵蚀文具的怪物，搜集资源并设法撤离。

这不是传统意义上的“单纯割草游戏”，也不是纯经营模拟。  
它更像是把三种体验焊成一台奇怪但带感的机器：

- **类幸存者式战斗成长**
- **搜打撤式风险收益抉择**
- **店铺经营与长期养成循环**

---

## ✨ 核心卖点

### 1. 文具魔法题材
不使用传统奇幻武器，而是把现实文具转译为战斗职责。  
铅笔负责精准刺击，纸张负责铺场召唤，橡皮负责清除与反制，圆规负责几何爆发，马克笔负责范围覆盖，订书机甚至能定身敌人。

### 2. 白天经营 + 夜晚战斗的双循环
玩家不是“在菜单里升级然后继续打”，而是在**店铺经营**与**异空间扫荡**之间来回切换。  
白天赚来的钱、夜晚带回的材料、店铺的侵蚀状态、武器强化与长期成长，会彼此咬合。

### 3. 类幸存者 + 搜打撤的融合
游戏中的每一局不只是“活得越久越强”，还要思考：

- 什么时候该继续贪
- 什么时候该撤
- 为了哪个流派锚点去冒险
- 带哪些物资回店最划算

### 4. 文具损耗具象化的敌人生态
敌人并非传统怪物，而是来源于现实中的**受潮、发霉、干涸、脆裂、锈蚀、褪色、失黏**等文具损耗。  
也就是说，你打的不是史莱姆和骷髅，而是“文具店版本的异常生态”。

## 🧭 核心玩法循环

```text
白天经营 → 准备物资 → 进入亚空间 → 战斗与收集 → 判断是否撤离 → 结算带出 → 回店整理与升级 → 下一日循环
```

### 单局体验包含

- 自动攻击 + 主动技能释放
- 地图探索与敌群处理
- 搜索资源点、获取材料与稀有物资
- 背包管理与风险抉择
- 寻找撤离点并完成带出

### 长线体验包含

- 店铺经营与商品售卖
- 新文具、新区域、新系统解锁
- 工坊成长与技能树推进
- 不同文具流派构筑
- 店铺侵蚀值管理

## 🌍 世界观概述

故事发生在原创世界中的 **磁核市（CEE KORE CITY）** 与 **蔚兰群岛**。

- **磁核市**：高密度、现代化、偏都市核心区
- **蔚兰群岛**：相对边缘、偏缓冲、接近异常与亚空间的区域
- **海域**：现实与非现实之间的过渡带，是雾气、异常与亚空间成立的重要媒介

玩家所经营的文具店位于蔚兰群岛。  
随着侵蚀现象不断扩大，主角逐渐卷入“现实店铺经营”与“异空间秩序失衡”交织的危机之中。

## 👾 敌人系统

敌人是由现实中文具与店铺环境的损耗、污染、老化所形成的亚空间具象化产物。

### 敌人类别

- 纸张系
- 霉菌系
- 墨水系
- 橡胶系
- 塑料系
- 金属系
- 胶黏剂系
- 木质系
- 颜料系
- 布料系
- 环境系

### 部分代表敌人

- **墨晕团**：受潮纸张与扩散墨迹形成的污染型敌人
- **脆页偶**：纸张脆裂后形成的高风险近战敌人
- **干涸鬼**：由失水笔类具象化而成的削弱型敌人
- **裂木狼**：木杆干裂形成的高威胁精英
- **灰尘侯**：由长期积尘形成的环境精英

### 敌人设计特点

- 每类敌人都有现实中的损耗来源
- 每类敌人都对应店铺经营中的负面后果
- 战斗结果会影响第二天经营状态

## 🎒 搜索、背包与撤离

游戏的“撤离”不是结算按钮，而是核心玩法之一。

### 搜索机制

- 普通物资与高价值物资需要不同时间收集
- 收集过程中敌人不会暂停
- 角色仍可低速移动并保留自动攻击能力

### 背包机制

- 背包容量有限
- 需要在材料、污染物、订单物品之间做取舍
- 可快速抛弃低价值物资，保持节奏流畅

### 双层撤离机制

- **主动撤离**：找到特定锚点并成功撤离，带回高比例资源并获得次日流派加成
- **时间满额撤离**：坚持到时间结束自动撤离，收益较稳但较低
- **失败撤离**：大部分资源损失，仅保留少量基础收益

### 六类文具锚点

不同锚点对应不同流派收益，例如：

- 记录锚点：提升次日精准 / 暴击
- 载体锚点：提升召唤上限
- 辅助锚点：缩短技能冷却
- 尺规锚点：提升范围伤害
- 画笔锚点：延长属性效果
- 收纳锚点：提升背包上限

## 🏪 经营系统

战斗带回来的资源不会只是“数字”，而是会真正进入店铺循环。

### 经营内容包括

- 商品售卖
- 商品定价
- 顾客偏好与随机来访
- 店铺陈列与装修
- 店铺功能升级
- 新品类解锁

### 经营与战斗联动

- 战斗失败会导致对应品类商品受损
- 资源可用于修复、补货、制作和升级
- 侵蚀值会持续影响店铺状态
- 店铺发展会反向增强玩家下一局的探索能力

---

## 📈 成长系统

游戏采用三层成长结构，而不是一棵臃肿的传统天赋树。

### 1. 工坊永久成长

局外长期升级，强化搜索、生存、撤离、经营等基础能力。

### 2. 文具流派专精

六类文具各自拥有独立专精方向，强化风格而非纯叠数值。

### 3. 局内临时构筑

每局通过升级与随机强化形成临时 build，承担类幸存者玩法的主要爽点。

这种结构让游戏同时具备：

- 长期投入回报
- 流派专精深度
- 单局构筑变化
